# 👨‍💼 Employee Management System (Task-02)

## 📌 Internship Task
This project is submitted for **Task-02: Employee Management System (CRUD Application)** under Prodigy Infotech Internship.

---

## 🚀 Project Overview
This is a full-featured **Employee Management System** developed using Laravel and MySQL.

The system allows administrators to efficiently manage employee records, track attendance, assign tasks, and perform CRUD operations while ensuring secure authentication and data protection.

---

## ✨ Features

### 🔐 Authentication & Authorization
- Secure login and registration system
- Role-based access (Admin, Employee, HR)
- Protected routes for sensitive operations

---

### 👥 Employee Management (CRUD)
- ➕ Add new employees  
- 📄 View employee details  
- ✏️ Update employee information  
- ❌ Delete employee records  

---

### 📊 Dashboard
- Overview of employees and activities
- Quick access to system modules

---

### 🕒 Attendance & Leave Management
- Track daily attendance
- Manage leave requests

---

### 📌 Task Assignment
- Assign tasks to employees
- Track progress and updates

---

### 💰 Payroll Management
- Manage salaries and payroll details
- Automate calculations

---

### 📈 Reports & Analytics
- Generate reports for better decision-making

---

## 🧩 Tech Stack

- **Backend:** Laravel (PHP)
- **Frontend:** Blade Templates, Bootstrap, Tailwind CSS
- **Database:** MySQL
- **Tools:** Composer, Vite

---

## 📂 Project Structure
```
/app → Core application logic  
/routes → Web routes  
/resources → Views (UI)  
/database → Migrations & seeders  
/public → Public assets  
```

---

## ⚙️ How to Run the Project

### 🔹 1. Clone Repository
```
git clone <your-repo-link>
cd employee-management-system
```

---

### 🔹 2. Install Dependencies
```
composer install
npm install
```

---

### 🔹 3. Setup Environment
```
cp .env.example .env
```

Update database credentials in `.env`

---

### 🔹 4. Generate App Key
```
php artisan key:generate
```

---

### 🔹 5. Run Migrations
```
php artisan migrate --seed
```

---

### 🔹 6. Start Server
```
php artisan serve
```

Open:
```
http://localhost:8000
```
